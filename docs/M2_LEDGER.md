# M2：持仓账本

账本模型分为账户、标的、持仓和交易四类记录。东方财富 A 股、东方财富港股通和 Revolut 使用独立账户 ID，因此既能按平台汇总，也能分别查看。

持仓保存数量、平均成本、成本币种、已知价格、价格币种、数据时间和来源。买卖交易额外保存实际成交数量、单价、总额、币种、费用和成交时间。

估值规则：

- 所有组合结果换算到账本的 `base_currency`；
- 汇率参数表示“一单位原币等于多少基准币”；
- 缺价格或汇率时，`market_value` 返回 `null` 并列出缺失项；
- 未知持仓不进入已知市值的占比计算，结果同时标记 `complete: false`；
- 集中度按账户、平台、市场、资金桶、币种和标的计算；
- 计算结果不写回原始账本，避免把派生值误当成用户录入事实。

真实账本默认位于 `data/private/ledger.json`，使用修订号和原子写入。
