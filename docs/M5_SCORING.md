# M5：指数水温与个股评分

## 指数水温

指数评分版本为 `index-temperature-v1`，输出范围为 0–100：0 表示相对冷，100 表示相对热。高温不是对未来下跌的预测，M6 只能把它作为新增资金的有限倾斜因素。

子项包括一年、三年、五年价格区间位置、距离高点回撤、估值分位、六个月和十二个月趋势、波动率、短期下跌速度及组合低配/超配。输出保留每项原值、分数、解释和权重。

## 观察名单个股

个股评分版本为 `watchlist-stock-v1`，只用于用户人工维护的观察名单。子项包括收入、利润、经营现金流、自由现金流、ROE、负债、利息保障、估值、趋势和波动。

审计否定/无法表示意见、退市风险、重大违规或持续经营风险属于独立红线。红线不参与加权平均，而是直接阻止推荐。

## 时间与数据质量

- 历史评分只读取 `published_at <= as_of` 的数据；财报按发布日期而非报告期判断可用性；
- 输出记录被排除的未来数据数量；
- 价格、市场或财务数据过期会停止推荐；
- 关键子项不足时置信度从 `high` 降为 `medium`、`low` 或 `stopped`；
- 每条评分连同版本和历史时点可写入本地评分记录。
