# M4：候选池与基金筛选

候选池先定义所需指数或资产暴露席位，再比较用户账户实际可买的产品。筛选不使用近期收益排名。

硬性排除包括：暴露不一致、不可购买、限额不足、杠杆/反向/复杂结构、关键数据缺失、规模或流动性不足，以及 QDII 溢价超阈值。

通过硬条件的产品按固定版本 `candidate-quality-v1` 评分：

| 维度 | 权重 |
| --- | ---: |
| 跟踪质量 | 30% |
| 年度成本 | 25% |
| 规模 | 15% |
| 流动性 | 15% |
| 产品历史 | 5% |
| 操作便利度 | 10% |

按总分降序、产品 ID 升序打破同分，保证输入顺序变化不影响结果。每个席位最多选择一个主产品和一个备用产品，没有合格产品时允许空缺。

每次初始、季度、产品变化、可购买状态或风险事件复核都会写历史；只有结果变化时才增加候选池版本。
