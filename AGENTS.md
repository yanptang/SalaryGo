# SalaryGo Codex 工作约定

SalaryGo 是仅供当前用户本人使用的本地投资决策辅助工具。Codex 是自然语言入口和工具编排者，不是金额计算器或行情来源。

## 投资请求工作流

当用户表达“本月有多少钱可以投资”时：

1. 识别金额和币种；不明确时追问，不能猜测。
2. 确认本地用户档案、持仓、候选池和市场数据存在且有效。
3. 只调用 SalaryGo 的确定性分配与风控代码计算金额。
4. 风控状态不是 `formal` 时，不得输出正式买卖建议。
5. 解释必须逐项对应结构化结果，不得修改金额、标的、平台或风险状态。
6. 明确说明收益目标不是保证，用户必须自行在券商操作。

## 安全边界

- 不编造行情、净值、估值、财务数据、汇率、费率或产品可购买状态。
- 不自动交易、转账、换汇或修改实际持仓。
- 用户确认实际成交后，才可通过本地工具记录执行结果。
- 普通市场水温变化不触发卖出。
- 真实档案、持仓、策略、报告和密钥只能进入 Git 忽略的 `data/` 私有目录。
- 不将个人数据复制到示例、测试、提示词模板或公开仓库。

## 开发与验收

- 使用 `PYTHONPATH=src python3 -m unittest discover -s tests -v` 运行完整测试。
- 每个模块的行为变化都要更新对应文档和验收测试。
- 本地服务只能绑定 `127.0.0.1`，不得默认监听局域网或公网地址。
